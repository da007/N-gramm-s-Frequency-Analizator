N-gramm's Frequency Analizator - программа, которая позволяет сделать статический анализ частотности N-грамм на основе представляемого текста. Учитывается частота относительно уникальных слов, с упором на составления [[Буквенно-Образные Коды|БОК]].

## Функции
- Анализировать уникальные слова(в идеале уникальность на основе лемм) и составлять таблицу частотности N-грамм.
- Выбирать оптимальный набор для составления [[Буквенно-Образные Коды|БОК]] на основе определенных слов(например: для слов состоящие из более 5 символов)

## Технологии
- Language: Python 3.12
- Frameworks: NLP, Tkinter
